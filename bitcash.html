BITCASH

I worked in a team of five to develop a service that allows users to send bitcoins via email, similar to Square Cash for Bitcoin. This project was produced at LA CodeDay, where our team won Best Application and Best SendGrid API integration.

I specifically worked on developing front-end design and development of the responsive splash page as well as back-end integration with the SendGrid API.

Technologies used: Node.js, SendGrid API, HTML5, CSS3, JavaScript, jQuery, Bootstrap
